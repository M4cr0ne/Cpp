VARIABILI

1. Idea chiave

Una variabile è uno spazio in memoria a cui diamo un nome. In quel “contenitore” possiamo mettere un valore, per esempio un numero, una lettera, una parola o un risultato calcolato.

Il programma usa le variabili per ricordare dati e lavorarci sopra. Senza variabili, il codice non potrebbe reagire agli input, eseguire calcoli o conservare informazioni.

In pratica: una variabile è come una scatola etichettata. L'etichetta è il nome della variabile, il contenuto è il valore memorizzato.

2. Modello mentale

Immagina tre scatole con etichette diverse: una per l'età, una per il prezzo, una per il nome. Ogni scatola può contenere un tipo di dato adeguato.

Esempio:

La variabile deve avere un identificatore riconoscibile.

Il tipo decide che tipo di valore può contenere.

Il contenuto può essere letto e modificato durante il programma.

3. Dichiarazione e assegnazione

La forma base di una variabile è questa:

tipo nomeVariabile;

Se vuoi anche assegnare subito un valore:

tipo nomeVariabile = valore;
Dichiarazione

int eta;

La variabile esiste, ma qui non le hai ancora dato un valore esplicito.

Inizializzazione

int eta = 17;

La variabile viene creata e valorizzata subito.

4. Tipi principali

Tipo Contenuto Esempio
int numeri interi int eta = 17;
float numeri con virgola float prezzo = 19.99;
double numeri con virgola più precisi double pi = 3.14159;
char un singolo carattere char iniziale = 'A';
bool vero o falso bool acceso = true;
string testo string nome = "Anna";
Scegliere bene il tipo è fondamentale: il tipo stabilisce che genere di informazione stai rappresentando.

5. Esempi guidati

Esempio 1 • Variabile intera

int eta = 17;

Qui eta contiene un numero intero.

Esempio 2 • Variabile con testo

string nome = "Marco";

Qui nome contiene una sequenza di caratteri, cioè un testo.

Esempio 3 • Modifica del valore

int punteggio = 10;
punteggio = 25;

Una variabile può cambiare contenuto durante l'esecuzione del programma.

Esempio 4 • Input dell'utente

int numero;
cin >> numero;

Qui la variabile riceve un valore inserito dall'utente.

Esempio 5 • Caso realistico: dati di uno studente

string nome = "Anna";
int eta = 17;
float media = 8.4;
Le variabili sono quasi sempre il primo strumento per rappresentare dati reali in un programma.

7. Errori tipici

Errore 1 • Dimenticare il tipo

In C++ non puoi scrivere solo il nome: devi dichiarare anche il tipo della variabile.

Errore 2 • Usare un nome non valido

Per esempio un nome con spazi o che inizia con un numero.

Errore 3 • Confondere = con “uguale matematico”

Nel codice = significa assegnazione: stai mettendo un valore nella variabile.

Errore 4 • Mescolare tipo e contenuto sbagliato

int nome = "Anna"; non ha senso: il tipo e il valore non sono coerenti.

Errore 5 • Pensare che la variabile sia il valore

La variabile è il contenitore con un nome. Il valore è il contenuto che può cambiare.

10. Riepilogo finale

  • Le variabili servono a memorizzare dati.
  • Ogni variabile ha un tipo, un nome e un valore.
  • Il tipo stabilisce che tipo di contenuto può avere.
  • Il valore può essere modificato durante il programma.
  • Le variabili sono alla base di input, calcoli, condizioni e strutture dati.
  • Capirle bene è essenziale per tutto il resto del C++.
Se uno studente capisce davvero bene le variabili, costruisce la base necessaria per comprendere quasi tutto il resto della programmazione.
